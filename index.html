Executive Summary (핵심 결론)

  • 올인원 한계 극복: 초소형 포장 코어(42ppm급)를 레버리지하고, 당사는 "도징 도크+레시피OS+카트리지/라이너+세정"에 집중하는 플랫폼 전략.

  • 현실적 성능 하한: 상용 레퍼런스 기준 50 packs/min, 93~186포/런, 최소 0.5g ±5% 조건을 성능 기준점으로 제시.

  • 경제적 세정 3층 구조: (1) 경로 분리/퀵체인지, (2) 오퍼레이터 트리거 수세, (3) 무소모품 에어블로우 및 건조 센서 확인. (FDA SOP 정합)

  • 고효율 AI 포지셔닝: 온디바이스 대형 모델이 아닌 스케줄링 최적화, LIW 파라미터 튜닝, 이상감지, 비전 잔류 확인에 집중하여 ROI 극대화.

LIW 제어 루프 (의사코드)

진동/기류/온도 격리와 디지털 필터링이 정확도에 핵심입니다.

# Pouch pulse dosing (LIW 기반)
start_w = loadcell_filtered()
target_w = start_w - target_mass_g

# Bulk 투입
while loadcell_filtered() > target_w + 5 * tolerance_g:
    set_rpm(HIGH)

# Trim (미세 조정)
while loadcell_filtered() > target_w + tolerance_g:
    set_rpm(LOW)

set_rpm(0)

# Verify (최종 검증)
dispensed = start_w - loadcell_filtered()
if abs(dispensed - target_mass_g) <= tolerance_g:
    PASS
else:
    FAIL  # 리젝트 / 재도징 / 수세 요구 트리거

  • 2

OOT (Out-of-Tolerance) 발생 시:

해당 포 자동 리젝트 → 동일 이벤트 재도징 → 연속 발생 시 채널 Lock 및 세정/수세 요구 (안전 인터록)

수세 모드 (Operator-triggered)

* FDA 세정 가이드 기반, 수용성 잔류 대응용 수동 트리거. (가동 중 분말+물 믹싱에 의한 페이스트화 원천 차단)

  • 인터록 전제: 카트리지 제거 확인, 호퍼 비움, 코어 비가동 상태 강제
  • 세척 대상: 도징 슈트, 버퍼 호퍼 내벽 등 교체 불가한 고정 부품
  • 기구 설계: 저압 분사 노즐, 배수 트레이, 내식성 R≥3mm 라운드 코너 구조
  • 프로세스: 수세 완료 후 "건조 모드"로 자동 전환 및 재가동 불가 유지

무소모품 건조 모드 (Dry Pass Gate)

소모품(실리카겔 등) 없이 상온 폐루프 에어블로우만으로 완전 건조를 수행하고, 다중 센서로 건조 증적을 남겨 규제/클레임을 방어합니다. (결로 회피 보장)

건조 합격(Dry Pass) 조건:
  • 배출 공기 RH(상대습도) 임계값 이하 안정화 (예: 35% RH 이하)
  • 슈트/호퍼 내벽 표면 온도 변화 수렴
  • 비전/광학 센서 상 물방울 패턴 60초 연속 미검출
Sensor Gate 온습도 + 비전 + 온도

1. Title

산제 자동 도징·포장 플랫폼: Recipe OS + 카트리지/라이너 기반 반복매출

2. Problem

약국 산제 업무 병목(인력·오류·세정) 및 레시피 다양성 운영 리스크 KPI 제시

3. Why Now

고령화/장기처방 증가, 인력난, 복약순응 포장 수요 증가

4. Solution

도징 도크 + 포장 코어 연동 + 수세/건조 + 레시피 OS 플랫폼화

5. Reference

상용 로봇 50ppm, 최소 0.5g ±5%, 자동세정 컨셉 존재를 "기술 하한"으로 제시

6. Architecture

모듈형 Scale-out과 3 SKU 로드맵(엔트리/스피드/정확도)

7. AI Moat

스케줄링·도징 튜닝·비전. LIW 진동 한계를 데이터 튜닝으로 극복

8. Quality & Safety

무소모품 건조 + 건조센서 게이트 + 2단 계량 + OOT 리젝트 (FDA 정합)

9. Business Model

HW + 구독 + 소모품. 포장재 지정/코어 임대 등 상용 락인 선례 활용

10. Go-to-market

레트로핏(기존 포장기) → 올인원(번들) → 센터필(멀티레인)

11. Unit Economics
